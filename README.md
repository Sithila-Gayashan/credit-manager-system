# Credit Manager System

A Java GUI-based Credit Management System with MySQL database integration.

## Features
- User Login/Registration
- Customer Management
- Bill Management
- Settlement Tracking
- Report Generation
- Bill Photo Upload

## Technologies
- Java Swing (GUI)
- MySQL (Database)
- iTextPDF (Reports)

## Setup Instructions

1. Install Java JDK 8+
2. Install MySQL and create database
3. Import `database/schema.sql`
4. Add required JAR files to lib folder
5. Run Main.java

## Database Configuration
Update DatabaseConnection.java with your MySQL credentials.
